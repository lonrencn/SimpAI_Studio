画布控制

点击 + 拖动平移画布视图
鼠标滚轮放大/缩小视图
Shift + 点击 (背景)开始调整画布区域大小
Shift + Ctrl + 点击开始移动整个画布
Shift + S + 左键点击绘制自定义输出区域形状
单击 (背景)取消选择所有图层
Esc关闭全屏编辑器模式

剪贴板 & I/O

Ctrl + C复制选定图层
Ctrl + V从剪贴板粘贴 (图像或内部图层)
拖放图像文件添加图像为新图层

图层交互

点击 + 拖动移动选定图层
Ctrl + 点击添加/移除图层选择
Alt + 拖动克隆选定图层
右键点击显示混合模式和不透明度菜单
鼠标滚轮缩放图层 (吸附到网格)
Ctrl + 鼠标滚轮精细缩放图层
Shift + 鼠标滚轮旋转图层 5°
Shift + Ctrl + 鼠标滚轮吸附旋转到 5° 增量
方向键移动图层 1px
Shift + 方向键移动图层 10px
[ 或 ]旋转 1°
Shift + [ 或 ]旋转 10°
Delete删除选定图层

变换手柄 (选定图层)

拖动角/边调整图层大小
拖动旋转手柄旋转图层
按住 Shift保持纵横比 / 吸附旋转到 15°
按住 Ctrl吸附到网格
