🎬 导演工作台

视频 Preset 分镜编排面板 —— 编辑分镜、管理素材、合成时间线。本文档面向维护者，说明当前实现与验证要点。

定位 界面结构 上传区语言 素材池 分镜规则 Preset 能力字段 代表性 Preset 生成行为 无限画布 上线验证

📌 当前定位

导演工作台 是视频 Preset 的分镜编排面板，与 高级参数 同级显示，不属于右侧窄参数区或某个 Preset 的普通参数组。

✅ 负责
  • 编辑多个分镜的开始、结束、提示词、图片/音频/视频引用
  • 把分镜转换成 prompt_override 和 director_timeline
  • 按 director_capability 限制素材类型、时长和串联方式
  • 逐分镜调用当前视频 Preset
  • 开启 Compose timeline 时合成最终视频
🚫 不负责
  • 替代 Preset 自己的模型参数
  • 替代 SAM3 蒙版、局部重绘、视频擦除等额外交互
  • 强制所有视频 Preset 支持导演模式

🖥️ WebUI 可见结构

展开 导演工作台 后，界面分为四块：

区域内容
顶部控制Director mode、Compose timeline
素材区图片 1–5、音频 1–5、视频 1–5
右侧栏合成宽度/高度/FPS、时间轴范围、素材规则说明、README 链接
分镜区分镜编辑器、图片引用按钮、音频/视频下拉、prompt 预览

宽屏布局

  • 左侧素材区，右侧整体侧栏（合成参数 + 规则说明紧密排列）
  • 分镜编辑器在素材区下方横向展开

窄屏布局

  • 顺序：顶部控制 → 素材区 → 右侧栏 → 分镜区 → prompt 预览
  • 不依赖浏览器 viewport 宽度，按导演台内容容器宽度响应

🌐 上传区语言

上传区固定文案由 javascript/scene_director.js 按当前语言刷新：

Images / 图片 Audio / 音频 Video / 视频 Click or drop / 点击或拖入 Clear / 清除 Director README / 导演台说明

语言来源优先级

  1. window.SimpAII18n.isEnglishUi(...)
  2. window.simpleaiTopbarSystemParams.__lang
  3. locale_lang
⚠️ 注意

Gradio 上传音频/视频后会重新替换素材区 HTML，固定文案不能只在初始化时翻译。
sceneDirectorRenderMediaPreview() 和 sceneDirectorRenderRules() 必须在每次刷新时重新写入当前语言。

💡 标记约定

Python 生成的固定 HTML 文案要保留 data-original-text 或 data-scene-director-* 标记，方便前端识别原始英文 key。

🎞️ 素材池

导演台共有 15 个素材位，与页面上方的 scene_canvas_image、scene_input_image*、scene_audio、scene_video 输入是分开的。导演模式生成时，当前分镜选中的素材会被映射到后端实际输入。

类型引用名说明
图片image_1 ~ image_5自绘拖入区，写入导演台自己的素材状态
音频audio_1 ~ audio_5通过隐藏 Gradio File 组件上传，再显示为卡片
视频video_1 ~ video_5通过隐藏 Gradio File 组件上传，显示缩略图
📋 当前策略
  • Preset 不支持的素材类型，区域仍显示但上传/拖拽/引用选择禁用
  • 导演台草稿按 Preset 分开保存；切换 Preset 加载对应草稿，不保证复用素材池
  • 已存在但当前 Preset 不允许的分镜引用会在前端和后端参数构建时自动清理

🎬 分镜规则

图片数量 → 分镜形态

图片数量形态输出
0Text-to-Video不写 @image
1Image-to-Video / first frame写入 1 个图片引用
2First / last frame写入首帧 + 尾帧
3–5Reference set写入参考图组

音频和视频规则

  • 每个分镜最多选择 1 个音频引用 + 1 个视频引用
  • 支持串联的 Preset 可选 previous_segment（上一段结果）
  • 是否允许由 director_capability 决定

分镜时长

  • 生成时长 = 结束 - 开始
  • 默认写入 scene_video_duration
  • Preset 可通过 segment_duration_param 改写目标参数名
  • video_duration_min/max 优先于旧 var_number_min/max

🏆 代表性 Preset

Preset能力概述
Wan(T2V)纯文本分镜，不使用任何素材引用
Wan(I2V)图片必需 最多 2 张，1 张首帧 / 2 张首尾帧
Wan-Extent视频必需 允许 previous_segment

InfiniteTalk音频必需 导演台校验音频素材
Wan-Animate / Wan-Remover当前不开放导演台

⚡ 生成行为

Director mode OFF

  • 不写 director_timeline，不写 prompt_override
  • 当前 Preset 按普通模式生成

Director mode ON

  • 前端把分镜写入隐藏 scene_director_editor_state
  • Python 侧构建 runtime，校验素材、图片数量、音频/视频必需项、时长范围
  • 逐分镜构建后端参数，映射媒体引用到后端输入
  • 分镜文本写入 prompt_override 和 director_prompt_override

Compose timeline ON

  • 分镜生成完成后，使用生成结果 + 音频策略 + 合成宽高/FPS 渲染最终时间线视频
  • Compose width/height/FPS 只影响最终合成，不改变单分镜模型参数
  • Timeline range 只影响预览轴和编排范围

手动检查清单

✓ 语言
  • 中文界面下上传区不出现 Images、Audio、Video、Click or drop
  • 英文界面下显示英文
  • 上传音频/视频后，标题和提示不会回到另一种语言
  • 右侧规则说明跟随语言切换
✓ Preset 能力
  • Wan(T2V) 不允许选择图片、音频、视频引用
  • Wan(I2V) 最多选择 2 张图片
  • LTX2.3 单分镜时长范围来自 Preset 声明
  • previous_segment 只在支持的 Preset 中可选
  • 不支持音频/视频时，素材区仍显示但入口禁用

生成失败排查

类别说明
模型缺失检查 models 目录
Comfy workflow 节点报错查看 ComfyUI 日志
Preset 能力声明不准确检查 director_capability 字段
WebUI 分镜参数构建错误检查 scene_director_webui.py
时间线合成阶段错误检查合成逻辑和 chain_output

🔧 维护注意

  • 固定文案统一放在 javascript/scene_director.js 的 SCENE_DIRECTOR_TEXT
  • Python 输出的 HTML 固定文案要加 data-original-text 或 data-scene-director-* 标记
  • 右侧栏由 #scene_director_side_panel 承载，不要把规则说明放回独立 grid 行
  • 媒体区域禁止交互时保持可见，避免用户误以为上传区消失
  • capability 字段变更后，同步检查 Preset JSON、WebUI、Infinite Canvas、release matrix 和 contract 测试
